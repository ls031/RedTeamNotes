 
#### _用nmap的情况_
###### 主机发现
```
nmap -sn 192.168.2.0/24
```
![nmap主机发现](img/2026-07-16-003657_2560x1600_scrot.png)
###### TCP端口扫描
```
sudo nmap --min-rate=10000 -p- 192.168.2.33
```
![TCP全端口扫描](img/2026-07-16-002845_2560x1600_scrot.png)

###### UDP扫描
```
sudo nmap -sU --min-rate=10000 -p- 192.168.2.33
```
![UDP全端口扫描](img/2026-07-16-002806_2560x1600_scrot.png)

###### 脚本扫描
```
sudo nmap --script=vuln -p22,111,139,445 192.168.2.33
```
![脚本扫描](img/kali-linux-2026.1-vmware-amd64-2026-07-16-12-59-16.png)
